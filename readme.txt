安装说明:

1:环境：windows/LINUX/UNIX/apache,mysql,php
2:所用语言: php,javascript,xml,html
3:将程序放置空间或者服务器上，要求uploadfiles目录可写,cache目录可写,common目录可写,config目录可写,data目录可写,templates_c目录可写,API目录可写
4:运行/install,根据提示输入相关信息,后台登陆地址为http://您的域名/admin
后台密码为您在运行install安装程序之初设定的email和密码,若忘记密码可咨询tgroupon官方重置方法.
5:前台访问方式为:http://您的域名，如果是二级目录可以http://您的域名/tg程序目录
